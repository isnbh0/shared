# Question Answering Skill

When you answer a question, identify the precise factual answer from the provided context.

Formatting rule: put ONLY the concise final answer inside `<answer>` and `</answer>` tags. The answer should be as brief as possible — typically a single word, number, or short phrase (e.g., a year, a name, a quantity). Do not write full sentences, restate the question, or add explanatory context inside the answer tags.

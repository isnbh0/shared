---
name: askme
description: Shorthand that tells the agent to ask clarifying questions instead of making decisions on its own
---

If other `/commands` appear in the user's message and you have not already called the Skill tool for them in this conversation, invoke each now. Do not re-invoke any skill that has already been loaded.

When you encounter ambiguities or decision points — before starting or mid-task — stop and ask the user to clarify or decide. Do not assume.
